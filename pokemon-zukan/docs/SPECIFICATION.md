# ポケモン図鑑 仕様書

## 1. プロジェクト概要

### 目的
フロントエンドのフェッチ機構を学習するためのポケモン図鑑アプリケーション。

### 学習目標
- ページネーション付きデータ取得
- 動的ルーティングと詳細ページ
- 検索機能とdebounce
- フィルタリング
- ローディング状態管理
- エラーハンドリング

### 技術スタック
| 技術 | バージョン | 用途 |
|------|-----------|------|
| Next.js | 16.x | フレームワーク（App Router） |
| React | 19.x | UIライブラリ |
| TypeScript | 5.x | 型安全性 |
| Tailwind CSS | 4.x | スタイリング |

---

## 2. 使用API

### PokeAPI
- **ベースURL**: `https://pokeapi.co/api/v2`
- **認証**: 不要
- **レート制限**: なし（常識的な範囲で）
- **ドキュメント**: https://pokeapi.co/docs/v2

### エンドポイント

| エンドポイント | 用途 | 例 |
|---------------|------|-----|
| `/pokemon` | ポケモン一覧 | `/pokemon?limit=20&offset=0` |
| `/pokemon/{id or name}` | ポケモン詳細 | `/pokemon/25` or `/pokemon/pikachu` |
| `/type` | タイプ一覧 | `/type` |
| `/type/{id or name}` | タイプ別ポケモン | `/type/electric` |

## 5. データ構造

### 5.1 型定義

```typescript
// ポケモン一覧のレスポンス
type PokemonListResponse = {
  count: number;
  next: string | null;
  previous: string | null;
  results: {
    name: string;
    url: string;
  }[];
};

// ポケモン詳細
type Pokemon = {
  id: number;
  name: string;
  height: number;  // デシメートル単位
  weight: number;  // ヘクトグラム単位
  types: {
    slot: number;
    type: {
      name: string;
      url: string;
    };
  }[];
  stats: {
    base_stat: number;
    stat: {
      name: string;
    };
  }[];
  sprites: {
    front_default: string;
    other: {
      'official-artwork': {
        front_default: string;
      };
    };
  };
};

// 一覧表示用に簡略化したポケモンデータ
type SimplifiedPokemon = {
  id: number;
  name: string;
  japaneseName: string;
  types: string[];
  imageUrl: string;
};

// 詳細表示用
type PokemonDetail = SimplifiedPokemon & {
  height: number;      // メートル単位に変換済み
  weight: number;      // キログラム単位に変換済み
  stats: {
    name: string;
    value: number;
  }[];
};
```

### 5.2 日本語名対応

PokeAPIの `/pokemon-species/{id}` エンドポイントから日本語名を取得可能。

```typescript
type PokemonSpecies = {
  names: {
    language: {
      name: string;
    };
    name: string;
  }[];
};
```

## 6. API関数設計

### 6.1 ファイル構成

```
src/
├── lib/
│   └── pokemon-api.ts    # API関数
├── types/
│   └── pokemon.ts        # 型定義
└── app/
    ├── page.tsx          # 一覧ページ
    └── pokemon/
        └── [id]/
            └── page.tsx  # 詳細ページ
```

### 6.2 関数一覧

| 関数名 | 引数 | 戻り値 | 説明 |
|--------|------|--------|------|
| `getPokemonList` | `offset`, `limit` | `SimplifiedPokemon[]` | 一覧取得 |
| `getPokemonDetail` | `id` | `PokemonDetail` | 詳細取得 |
| `searchPokemon` | `query` | `SimplifiedPokemon[]` | 名前検索 |
| `getPokemonByType` | `type` | `SimplifiedPokemon[]` | タイプ別取得 |
| `getTypeList` | - | `string[]` | タイプ一覧取得 |
